Color Opacity Tokens

This demo shows how to apply opacity to theme colors using utility classes. The examples below use bg-{{token}}-{{step}} to set background color with a given opacity. You can also use the same pattern for:
  • text-{{token}}-{{step}} — text color with opacity
  • border-{{token}}-{{step}} — border color with opacity
All of these are theme‑aware and will update automatically when the theme changes.
