BASE Color Tokens

The tokens below represent the base theme colors available in NovixEngine. Each token is exposed as a CSS variable (e.g., var(--novix-primary)) and updates automatically when the active theme changes (light/dark or custom themes). NOTE: the '--novix' prefix is only used in the underlying variables. Any created classes are built to follow Bootstrap-style syntax to follow a common world structure.

  • primary - Main brand/action color. Example: background-color: var(--novix-primary)
  • secondary - Secondary brand/accent color.
  • accent - Highlight or call-to-action color.
  • neutral / neutral-variant - Grays and supporting tones for backgrounds/borders.
  • on-* tokens - Text/icon colors that contrast with their paired background token.

Use these tokens in your component SCSS or inline styles with var(--novix-token-name). For example: .btn-primary { background-color: var(--novix-primary); color: var(--novix-on-primary); }
