BASE Color Tokens

The tokens below represent the base theme colors available in NovixEngine. Each token is exposed as a CSS variable (e.g., var(--primary)) and updates automatically when the active theme changes (light/dark or custom themes).

  • primary - Main brand/action color. Example: background-color: var(--primary)
  • secondary - Secondary brand/accent color.
  • accent - Highlight or call-to-action color.
  • neutral / neutral-variant - Grays and supporting tones for backgrounds/borders.
  • on-* tokens - Text/icon colors that contrast with their paired background token.

Use these tokens in your component SCSS or inline styles with var(--token-name). For example: .btn-primary { background-color: var(--primary); color: var(--on-primary); }
