Novix Cabinet Component

The Novix Cabinet is a flexible UI component that can be anchored to any edge of the user's screen — top, bottom, left, or right — with the default position set to the left. It provides a modular system of slide-out trays, each activated by its own handle. These trays can contain independent content, making the Cabinet ideal for organizing tools, panels, or contextual features in a compact, accessible layout.
