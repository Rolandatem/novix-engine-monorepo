CSS Overriding Demo

If you’d prefer to adjust the Novix Cabinet’s appearance using CSS instead of override properties, you can use the following variables.

Cabinet and Tray CSS Variables
Variable
Description
--novix-cabinet-tray-handle-background-color
Defines the background color of tray handles. -primary
--novix-cabinet-tray-handle-color
Sets the text color used on tray handles. -on-primary
--novix-cabinet-tray-handle-font-family
Specifies the font family for tray handle text. -font-family
--novix-cabinet-tray-handle-font-size
Controls the font size of tray handle text. -font-size-xs
--novix-cabinet-tray-content-background-color
Defines the background color of the tray’s content area. -surface-variant
--novix-cabinet-tray-content-border-color
Sets the border color of the tray. -primary
