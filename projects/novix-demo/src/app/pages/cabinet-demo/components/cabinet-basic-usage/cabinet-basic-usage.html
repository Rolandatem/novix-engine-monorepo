To get started with the Novix Cabinet, first import the NovixCabinetModule into your Angular component.

Once available, add the <novix-cabinet> element to your template. Inside the cabinet, define one or more trays by using the <ng-template novix-cabinet-tray> template and directive for each tray you want to include.

Each tray should provide a handle label so users can identify and interact with it.

This setup gives you a fully functional cabinet with customizable trays -- the simplest way to start using Novix Cabinet in your application.

IMPORTANT NOTE: When using a vertical and a horizontal cabinet there will be overlapping at the corner that they meet. Make sure to either using edgMargin or justifyHandles so that all handles are clickable.
