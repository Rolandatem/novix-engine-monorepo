Properties

The following optional properties let you customize how the Novix Cabinet looks and behaves. Use them to adjust its appearance or tweak its functionality to better fit your needs. Bolded options are the default values.

Cabinet Properties
Property
Type
Options
Description
attachDirection
string
top | left | bottom | right
Specifies which edge of the screen to cabinet should attach to when displayed.
rounded
boolean
true | false
Enabled rounded corners on tray handles for a softer visual appearance.
autoCloseOnOutsideClick
boolean
true | false
Automatically closes an open tray when the user clicks outside of the cabinet area.
edgeMargin
string
Margin (in pixels) applied to the top/bottom or left/right edges.
justifyHandles
string
start | center | end
Controls how tray handles are horizontally aligned.
