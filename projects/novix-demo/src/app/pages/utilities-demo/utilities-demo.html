Novix Engine CSS Utilities

NovixEngine provides a suite of Bootstrap-style utility classes for layout, spacing, display, and more. These utilities are responsive, theme-aware, and designed for developer familiarity.

üìê Spacing Utilities

Use .m-* and .p-* classes to control margin and padding. Responsive variants like .m-sm-4 adapt across breakpoints.

.p-3 .m-2
.p-sm-4 .m-md-5
ClassDescription
.m-1Margin: 1 unit (0-4)
.p-2Padding: 2 units (0-4)
.mx-3, .pt-1Directional: (t: top, b: bottom, s: start, e: end, x: start+end, y: top+bottom)
Responsive options available (sm: small, md: medium, lg: large, xl: extra large)
Examples: .m-sm-2 | .p-sm-1
